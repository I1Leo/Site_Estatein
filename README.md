# Estatein

Был разработан сайт агентства недвижимости Estatein.

Сайт написан на React с использованием TypeScript. Для управления состоянием используется Redux Toolkit. Сборка проекта осуществляется с помощью Vite.

## Страницы

Сайт состоит из следующих страниц:

- **Главная страница** — содержит hero-секцию с описанием агентства, блок с ключевыми преимуществами, слайдер избранных объектов недвижимости, отзывы клиентов и раздел с часто задаваемыми вопросами.
- **О нас** — рассказывает о ценностях компании, достижениях, опыте работы, команде и клиентах агентства.
- **Недвижимость** — включает форму поиска объектов с фильтрами, слайдер доступных объектов и форму обратной связи.
- **Услуги** — описывает предоставляемые сервисы: оценку недвижимости, маркетинг, управление объектами и инвестиционное сопровождение.
- **Контакты** — содержит информацию об офисах компании, форму для связи и галерею.

Роутинг осуществляется с помощью React Router DOM.
